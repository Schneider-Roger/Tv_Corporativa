# TV Corporativa

Este repositório contém o design inicial de um sistema de TV corporativa para múltiplas cidades, unidades e setores.

## Objetivo

Gerenciar a distribuição de conteúdo para telas corporativas com hierarquia geográfica e organizacional:

- **Cidade** → **Unidade** → **Setor** → **Telas**

## Próximos passos

1. Validar o modelo de dados e os fluxos de publicação.
2. Definir permissões por papéis (administrador global, gestor de cidade, gestor de unidade, editor de setor).
3. Implementar o backend e o painel administrativo.

Consulte a documentação em `docs/` para detalhes do modelo.
